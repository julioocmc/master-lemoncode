# Ejercicio 1 - Paletas dinámicas con Sass

## Descripción

Este ejercicio crea múltiples paletas de colores dinámicas utilizando Sass.

A partir de varios colores base se generan automáticamente:

- 4 tonos más oscuros
- color base
- 4 tonos más claros

Cada paleta se genera dinámicamente utilizando:

-> Variables
-> Maps
-> Mixins
-> Functions

El color del texto se ajusta automáticamente para mantener buena legibilidad dependiendo del fondo.
